== Description ==

Imports products to your group in ok.ru from your online store. The plugin Woocommerce is required!

PRO version: [https://icopydoc.ru/product/import-products-to-ok-ru-pro/](https://icopydoc.ru/product/import-products-to-ok-ru-pro/?utm_source=wp-repository&utm_medium=organic&utm_campaign=import-products-to-ok-ru&utm_content=readme&utm_term=pro-version)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire `import-products-to-ok-ru` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to `Set API ok.ru` and follow the instructions to configure API access
1. Use the `Export to OK.RU` screen to configure the plugin
